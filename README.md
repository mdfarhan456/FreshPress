# FreshPress

FreshPress is a modern React-based news web application that provides real-time news updates using an external API. The project demonstrates routing, API integration, and clean UI design.

---

## Setup Instructions

1. Clone the repository:
  git clone https://github.com/mdfarhan456/FreshPress.git

2. Navigate to the project directory:
   cd FreshPress

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open the app in your browser (usually at http://localhost:5173)

---

## Module Overview

- Home  
  Displays a hero section with a welcome message and a button linking to the News page.

- About  
  Contains information about the developer (BCA graduate, currently pursuing MCA).

- News  
  Fetches and displays real-time news articles using the GNews API.  
  Each article includes a title, description, and a “Read More” link.

- Contact Us  
  Simple contact form with an alert message on submission (no backend).
